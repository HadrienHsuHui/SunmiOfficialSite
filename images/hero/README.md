# Hero Images

This directory contains hero and feature images for the SUNBAY website.

## Image Files

1. **hero-main.png** - Main hero image for homepage
2. **grocery-checkout.jpg** - Grocery store checkout scene (retail page)
3. **developer-coding.png** - Developer coding scene (ISV/developer pages)
4. **terminals-display.png** - Payment terminals display (hardware/product pages)
5. **popup-store.webp** - Pop-up retail store (retail solutions)
6. **restaurant-service.jpg** - Restaurant service scene (restaurant page)

## Usage

These images are referenced in the following pages:
- index.html → hero-main.png
- retail.html → grocery-checkout.jpg, popup-store.webp
- restaurant.html → restaurant-service.jpg
- startup-iso.html → developer-coding.png
- hardware.html → terminals-display.png

## Image Requirements

- Format: JPG, PNG, or WebP
- Recommended size: 1920x1080 or higher
- Optimized for web (compressed)
